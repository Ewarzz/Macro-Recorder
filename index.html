// Features

Everything you need.

01 ⏺
Record anything

Captures every mouse move, click and keypress with millisecond accuracy. Just press the record key and go.

02 ✦
Visual Macro Editor

Build or fine-tune macros block by block — key presses, mouse clicks, delays. No recording needed.

NEW
03 ⌨
Custom Trigger Keys

Assign any keyboard key or mouse button to trigger a specific macro — independent from global hotkeys.

NEW
04 ⏸
Hold to Repeat

Enable hold mode per macro — the macro loops while your trigger key is held and stops the moment you release it.

NEW
05 ◉
Toggle Mode

One press starts, one press stops. Perfect for long sequences you don't want to hold down.

NEW
06 ⊘
Global ON / OFF

Instantly disable all macro trigger keys with one button. Safe to type without accidental triggers.

NEW
07 ⚡
Speed Control

Play back at 0.1× or 100× original speed. Full control over timing without re-recording.

08 🎮
DirectInput Compatible

Uses Windows SendInput API for mouse events. Works in games that block standard input simulation.

09 🌐
English & French

Full interface in English (default) and French. Switch language instantly from the settings tab.
